# Campus ambassador program — 1 scout per college

**Labels:** `growth`, `community`, `priority: high`  
**Milestone:** Distribution

## Problem
Scrapers miss WhatsApp-only events. One engaged student per campus is cheaper and more accurate than building 50 scrapers.

## Tasks
- [ ] Define scout role: post 2+ events/month from their campus, verify WhatsApp listings
- [ ] Recruit 1 person each at: RVCE, PES, MSRIT, BMSCE, IIIT-B (start with 5)
- [ ] Give scouts: auto-trust perk, public leaderboard credit, "Bangalore Scout" badge
- [ ] Create simple onboarding doc (how to use `/submit`, what counts as an event)
- [ ] Track submissions per college in admin UI
- [ ] Recognize top scout on homepage monthly

## Acceptance criteria
- At least 3 colleges have an active scout within 4 weeks of launch
- Each scout submits ≥1 verified event in first month
- Crowdsourced events ≥30% of feed for their college
