# Team-finding board — "need a designer for X hackathon"

**Labels:** `feature`, `community`, `priority: low`  
**Milestone:** Community & retention

## Problem
Students often have the hackathon link but no team. A board scoped to Bangalore + specific events fills a gap Unstop doesn't cover.

## Tasks
- [ ] Schema: `TeamPost` — eventId, role needed, skills, contact, college, expiresAt
- [ ] Page: `/teams` or per-event section "Find teammates"
- [ ] Filter by event, college, role (dev/design/business)
- [ ] OTP to post (reuse existing flow)
- [ ] Auto-expire posts after event starts
- [ ] Optional: link to college WhatsApp groups (manual URLs)

## Acceptance criteria
- User can post "need backend dev for PES hackathon this weekend"
- Posts tied to a specific event or general Bangalore hackathon
- Expired posts hidden automatically
